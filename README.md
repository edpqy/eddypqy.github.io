# Profile page files (static)

Upload these files to the **root** of your `edpqy.github.io` repo:
- index.html
- styles.css
- script.js

Optional:
- Upload your CV as `CV.pdf` (the sidebar CV button will work automatically)
- Update Glioportal portal link in `index.html` (search for the Glioportal 'Portal' button href)

This static site works even if Hugo is not building yet.
